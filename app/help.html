Attendance App Help Guide

Quick Start

  1. Log in with your assigned account.
  2. Select a group or keep All Groups when appropriate.
  3. Select Practice, Games, Team Events, or Players.
  4. Select the event or open Player Management.
  5. Save changes before leaving the page.

Roles

  • Admin: can manage players, create team events, manage rosters, and take attendance.
  • Team Mom: can manage players, create team events, manage rosters, and take attendance.
  • Main Coach: can view players and take attendance, but should not add or edit players.

Practice Attendance

  1. Select a birth-year group or choose All Groups.
  2. Open the Practice tab.
  3. Select the practice date.
  4. Use search, birth year, gender, and status filters to narrow the list.
  5. Tap Present, Absent, Excused, Cancelled, or Clear / Reset.
  6. Click Submit Attendance.

Game Attendance

Game rosters are managed one team at a time. Select a specific birth-year group before editing a Game roster.
  1. Select the specific group for the game.
  2. Open the Games tab.
  3. Select the game date.
  4. Use Manage Event Roster to choose the expected players.
  5. Save the roster, then continue to attendance.
  6. Use search, birth year, gender, and status filters as needed.
  7. Submit attendance.

Team Events

  1. Open the Team Events tab.
  2. Create a new Team Event when needed.
  3. Select the exact players for the event. Use Gender filter for girls-only scrimmages or special events.
  4. Save the Team Event.
  5. After the event is created, continue to roster review or attendance.

Player Management

  • Use Add New Player to open the add form.
  • Player cards are designed for quick lookup. They show player number, birth year, gender, status, Parent 1 name, phone, email, and Parent 2 contact if entered.
  • Use View Details or tap a player card to see the full read-only profile, including snack, paperwork, photo release, emergency contact, address, and last updated date.
  • Use Edit only when changing player information.
  • Use filters for birth year, gender, missing paperwork, missing photo release, and missing emergency info.
  • Use Make Inactive when a player should no longer appear in attendance lists.

Gender Field

  • Gender options are Male and Female.
  • Player cards show gender as M or F.
  • Gender filters are available in Player Management, Practice attendance, Game attendance, and Team Event player selection.

Event Actions

  • Cancel Event: use for rainouts or normal cancellations. This keeps event history.
  • Restore Event: use when a cancelled event needs to be reopened.
  • Delete Event: use only for mistakes or duplicate events. Delete is permanent.

Snack Rules

  • Bring Snack: family can be assigned snack duty.
  • Paid Out: player is tracked, but parent is skipped for snack duty and coach provides snacks.
  • Cancelled games should roll snack duty forward to the next game when snack scheduling is added.

Troubleshooting

  • If the app looks old after an update, hard refresh with Ctrl + F5.
  • If players do not show, confirm the correct group and event are selected.
  • For Games, confirm a roster has been saved before taking attendance.
  • If login fails, refresh and try again. If it continues, check that the backend API is online.
