Attendance App Help Guide

Quick Start

  1. Log in with your assigned account.
  2. Select Dashboard, Practice, Games, Team Events, or Players.
  3. Select the event or open Player Management.
  4. Save changes before leaving the page.

Roles

  • Admin: can manage players, create games and team events, manage rosters, and take attendance.
  • Team Mom: can manage players, create games and team events, manage rosters, and take attendance.
  • Main Coach: can view players and take attendance, but cannot add, edit, or create events.

Dashboard

  • The Dashboard is for quick reports, not player editing.
  • Use the month filter to review April, May, June, the current month, or all months.
  • Birthdays follow the selected month. If All Months is selected, birthdays show the current month and next month.
  • Monthly Attendance Summary shows total practices, games, team events, and attendance percentages.
  • Practice, Game, and Event summaries show totals, attendance percentage, players at 70% or lower, and players at 85% or higher.
  • Players Needing Attention, Perfect Attendance Club, and Outstanding Attendance are collapsible sections.
  • Player cards have clickable Practice, Game, and Team Event buttons to show details only when needed.

Practice Attendance

  1. Open the Practice tab.
  2. Select the practice date from the dropdown.
  3. Use search, birth year, gender, and status filters to narrow the list.
  4. Tap Present, Absent, Excused, Cancelled, or Clear / Reset for each player.
  5. Click Submit Attendance.

Game Attendance

Games use an exact player roster. You pick the specific players for each game when creating it, and the roster can be edited any time before or during the event.
  1. Open the Games tab.
  2. To create a new game: click + Add New Game, fill in the name/opponent, date, start time, and location, then select the expected players and click Save Game.
  3. To take attendance on an existing game: select the game from the dropdown. Only the rostered players will appear.
  4. If the roster needs changes before taking attendance: click Edit Roster. All active players will appear with the current roster pre-checked.
  5. Use the Edit Roster filters: search, birth year, gender, and roster view. Roster View can show all players, selected only, or not selected.
  6. Use search, birth year, gender, and status filters during attendance as needed.
  7. Mark each player and click Submit Attendance.

Team Events

  1. Open the Team Events tab.
  2. Click + Add New Team Event.
  3. Enter the event name and date. Select the exact players for the event. Use the Gender filter for girls-only or boys-only events.
  4. Click Add Team Event. The event is created and selected automatically.
  5. Only selected roster players should appear in Team Event attendance.
  6. Continue to attendance when ready.
  7. To edit the roster later: select the event and click Edit Roster. Edit Roster should show eligible players with selected players checked.

Player Management

  • Click Add New Player to open the add form.
  • Player cards show player number, birth year, gender, status, Parent 1 name, phone, email, and Parent 2 contact if entered.
  • Click View Details or tap a player card to see the full read-only profile, including snack preference, paperwork status, photo release, address, emergency contact, and last updated date.
  • Address and Emergency Contact are shown as separate sections for easier reading. Alt emergency phone stays in Edit Player but is not shown on the quick details card.
  • Use Edit only when changing player information.
  • Use filters for birth year, gender, missing paperwork, missing photo release, and missing emergency info.
  • Use Make Inactive when a player should no longer appear in attendance lists. Use Show inactive players to view or restore them.

Gender Field

  • Gender options are Male and Female.
  • Player cards show gender as M or F.
  • Gender filters are available in Player Management, Practice attendance, Game attendance, and Team Event player selection.

Event Actions

  • Cancel Event: use for rainouts or normal cancellations. Keeps event history and can be restored.
  • Restore Event: reopens a cancelled event and removes cancelled attendance rows so attendance can be marked again.
  • Delete Event: permanent. Use only for mistakes or duplicate events — not for normal cancellations.

Snack Rules

  • Bring Snack: family can be assigned snack duty.
  • Paid Out: player is tracked, but parent is skipped for snack duty and coach provides snacks.

Troubleshooting

  • If the app looks old after an update, hard refresh with Ctrl + F5.
  • If players do not show for a game, confirm the game roster has been saved. Click Edit Roster to check.
  • If players do not show for a practice, confirm the correct practice date is selected.
  • If login fails, refresh and try again. If it continues, check that the backend API is online.
  • If your session expires, the app should automatically return to the login screen and ask you to log in again.
